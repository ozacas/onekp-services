MAAB: datasets & methods

 

We describe the assembled transcriptomes produced for identification and analysis of the Hydroxyproline rich glycoprotein family (HRGP) and the methods for data analysis employed. 

 

 

Datasets

 

Each sample, over 1200 provided by the OneKP consortium, was assembled four times using a multiple-K methodology with K set to 39,49,59 and 69 consecutively. This leads to four transcriptomes per sample, one for each unique K-mer. Frequently, transcripts will be reconstructed by several k-mers. We make available each transcriptome using its four-letter OneKP-assigned sample ID:

Upon request, we can discuss large-scale data transfers but the full dataset is several terabytes (a terabyte is a million megabytes), so can take considerable time and effort to migrate.
